> Mind XX ACPR project

https://mind-xx-acpr.firebaseapp.com/

## Build Setup

```bash
# install dependencies
yarn install

# serve
yarn serve

# build for production with minification
yarn build

# storybook
yarn serve

# lint
yarn lint

# run jest
yarn test:unit

# run puppeteer
yarn test:pupp
```
